================= MANUAL DE USO - DIAGRAMA DE FLUJO AUTOMÁTICO =================

1. Ingresar los programas de c++14 como input (archivos txt) en la ruta del proyecto.
2. En el source -> main configurar la ruta del archivo de entrada que quiera testear.
3. Ejecutar el archivo main y disfrutar de un super diagrama de flujo automático.

Notas: 
	- Se recomienda usar el IDE Pycharm ya que ahí se contruyo el proyecto.
	- Usar python 3.7 y antlr 4 ya que se usaron para el proyecto.
	- No olvidar añadir como interpretador python 3.7 y antlr4-python3-runtime en 
	  settings -> project -> '+'
	- No olvidar añadir el plugin de antlrv4 para PyCharm.
